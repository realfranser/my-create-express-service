# My Create Express Service

Basic template to create an express service with Typescript

## Technologies

### Framework

Express

### Linting and code style

ESLint (Google rule set)
Prettier

### Package Manager

Yarn

### Error handling

Boom

## How to use

1. Install dependencies:
   > yarn
2. Run in dev mode:
   > yarn dev
3. Build project:
   > yarn build
4. Preview built project:
   > yarn preview
